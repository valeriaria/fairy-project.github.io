# Fairy widgets / Затишні віджети

Цей сайт пропонує вам естетичні мінімалістичні віджети для розміщення на вашому рабочому столі та створення затишного атмосферного ефекту.

# Віджети

1. **Цифровий Годинник**
   - Відображає поточний час у форматі години:хвилини:секунди. Оновлюється автоматично кожну секунду.

2. **Прогноз Погоди**
   - Надає інформацію про погоду для будь-якого міста. Просто введіть назву міста, і ви отримаєте актуальний прогноз погоди.

3. **Календар**
   - Відображає поточну дату та день тижня. Зручний та мінімалістичний.

# Використання

Просто відкрийте сайт та виберіть той віджет, який вам потрібний. Для більшого ефекту використовуйте декілька моніторів чи інших пристроїв відображення.
